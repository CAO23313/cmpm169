Experiment 1 - Neon kaleidoscope

Description

What are you trying to acomplish with this experiment?

I want to explore vector art in this experimental assignment, by those inspired artwork examples, I can create my own experiment artwork.

Technical

Describe in detail what you did technically and how it works?

I used two artwork examples and combined their code together for integration, and then added my own idea of a style kaleidoscope into the experimental artwork by editing the code. The program has three different modes that can be switched by pressing the number key 1,2,3. It will change the matrix to a different shape, and the scale of the matrix will be modified by the mouse pointer's position.
